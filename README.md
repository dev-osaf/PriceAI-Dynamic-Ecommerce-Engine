# PriceAI: Dynamic E-Commerce Pricing Engine

## Executive Summary
PriceAI is a multi-layered analytical ecosystem designed to automate and optimize e-commerce pricing strategies. By synthesizing supervised learning, evolutionary computing, and fuzzy logic, the platform replaces manual, static pricing with a data-driven, adaptive model that maximizes profitability while maintaining market competitiveness.

## 🔬 Core Methodology
The system architecture integrates five distinct AI pillars to ensure robust decision-making:

* **Dynamic Pricing Engine (FNN):** A Feedforward Neural Network trained to correlate market variables (seasonality, competitor activity, historical demand) with optimal price points.
* **Discount Strategy (Fuzzy Logic):** A soft-computing controller that navigates market uncertainty, providing granular discount recommendations that avoid the volatility of binary decision-making.
* **Time-Series Forecasting (LSTM):** An input-gated Recurrent Neural Network designed to predict transaction volumes across a 7-day rolling window.
* **Profit Optimization (Genetic Algorithm):** An evolutionary search algorithm that performs combinatorial optimization to pinpoint the "sweet spot" for net revenue.
* **Competitive Intelligence:** A clustering and classification pipeline (K-Means & Naive Bayes) used to profile market competitors and mitigate price-war risks.

## 🛠 Tech Stack
* **Language:** Python
